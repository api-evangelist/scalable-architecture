Oracle Cloud's donation of compute resources lets CNCF projects develop Arm64 support more effectively. With over 50% of new instances on AWS and over 33% on Azure running on Arm64 CPUs, multi-architecture support is essential, and projects use OCI-hosted GitHub Actions runners for native CI testing on Arm platforms.
