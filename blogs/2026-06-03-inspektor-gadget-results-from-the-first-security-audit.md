The CNCF project Inspektor Gadget has completed its first independent security audit, and the results are now published. The audit examined the eBPF-based tooling used for debugging and inspecting cloud native workloads running in Kubernetes clusters. Findings and remediation steps are detailed for teams using Inspektor Gadget in production scalable architectures.
